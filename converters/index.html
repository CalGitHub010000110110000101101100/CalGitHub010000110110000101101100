Convert wav to mp3

Use JPG Quality.

  1. Select your image file for "image to ruin," or paste an image using ctrl/cmd + V.
  2. Leave the "jpeg" dropdown as it is.
  3. Lower quality will have a smaller file size, but higher quality will make the image look less corrupted. Set it to 1 to not lose quality.
  4. Right click on the image that appears and save the image.
